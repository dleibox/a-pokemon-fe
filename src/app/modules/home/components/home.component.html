A Pokémon Angular App

based on Angular 9+, Angular Material and NgRx/Redux

What's included:

  • Angular 10+ & Typescript
  • Angular Material & SCSS
  • Responsive layout
  • NgRx (Store, State, Action, Reducer, Effect, Entity Adapter)
  • CI/CD (GitHub/Bitbucket, CircleCI pipeline, Docker, Test, SSH Deployment, Bash Script)
  • Nginx
  • Vultr Ubuntu VPS, Heroku Cloud Containers

All the Pokémon data are provided by PokeAPI
